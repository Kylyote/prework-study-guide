HTML

  • The head element of HTML holds info about the webpage such as font, scale, and other info read by the browser
  • The body element of HTML contains the content that will be seen by anyone viewing the webpage

CSS

  • CSS can be used to dictate properties to the whole sheet, through inline, internal, and external methods. External makes editing easier and spreading the style across the website
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content.
  • Sections of a page can be given labels that the CSS will control by giving a "class" in the declaration.

Git

  • git status: checks what branch is currently active
  • git checkout -b "branh-name: creates a new branch and makes it active
  • git checkout main: makes main branch active, will give warning if current active branch has not been committed
  • git add -A: commits files to working branch
  • git commit -m "commit note": m flag tells commit to add a note
  • git pull origin main: pulls base branch into feature branch, good habit to do before pushing new code
  • git push origin "name of feature branch": pushes feature branch to main repo
